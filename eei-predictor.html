MONTE CARLO UNCERTAINTY (N=500)

Each simulation run adds Gaussian noise (σ=5%) to all input parameters. After 500 runs, we compute the mean and 95% confidence interval (±1.96σ) for each scenario probability. This shows how sensitive the model is to input uncertainty.

SENSITIVITY ANALYSIS

Each input parameter is swept from 0→100% while others are held constant. The resulting probability range for each scenario quantifies that parameter's influence. Higher range = higher sensitivity.

LIMITATIONS

This is a parametric estimation model, not a causal simulation. It captures directional relationships between inputs and scenarios based on geopolitical analysis, but does not model feedback loops, non-linear phase transitions, or emergent phenomena. Treat outputs as structured scenario exploration, not prediction.
